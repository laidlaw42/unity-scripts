- Create a "Selectable" tag
- Import the QuickOutline asset from the Asset Store
- Create an empty object, name it "Selection Manager" then add the following scripts:
	- Selection Manager
	- Outline Selection Response
	- Ray Cast Based Tag Selector
	- Mouse Screen Ray Provider

- On the Player Controller camera, ensure Physics Raycaster is added AND tag the Camera as "Main Camera"

- For each item you wish to outline, add the QuickOutline "Outline" script and select a colour.
